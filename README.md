# APP快速打代理包工具


*前言：*
代理包可以快速帮助推广运营进行多渠道分发：  

 - 实现多渠道打包及iOS企业级分发包的快速生成渠道包
 - 只需要一个Android iOS原始包即可快速生成N个渠道商代理包
 - 支持生成之后快速扫二维码下载该渠道包测试
 - python单文件GUI开发。方便拆解及替换app内文件，及功能模块单独使用
